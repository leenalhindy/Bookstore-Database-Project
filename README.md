# BookStore Database Management System 

A relational database project built using MySQL to manage a bookstore inventory. This project demonstrates database design, data integrity, and complex querying.

## Project Structure
- `scheme.sql`: Contains the database creation and table structures (DDL).
- `data.sql`: Contains the initial data population (DML).
- `queries.sql`: Includes complex joins, aggregations, and stored procedures.

## Key Features
- Relational design with Primary and Foreign keys.
- Data integrity constraints (NOT NULL, UNIQUE).
- Advanced SQL queries and automation.
